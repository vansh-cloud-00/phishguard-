ℹ️ ABOUT THIS PROJECT

PhishGuard

A Phishing Awareness & Cyber Safety Learning System

Built to Keep You Cyber Safe

PhishGuard is an academic project designed to educate everyday internet users about phishing attacks through interactive learning, quizzes, and threat awareness tools. Our goal: make cybersecurity education accessible, engaging, and practical for everyone.

🎯

PROJECT PURPOSE

Phishing is responsible for over 90% of data breaches globally, yet most people cannot identify a phishing attack. PhishGuard bridges this education gap through a practical, interactive platform that teaches users to recognize, understand, and respond to phishing attempts.

This project was built as an academic front-end development exercise demonstrating real-world application of HTML, CSS, and JavaScript without any external frameworks.

⚙️

TECHNICAL OVERVIEW

Built entirely with vanilla web technologies — no frameworks, no dependencies.

HTML5 CSS3 Vanilla JavaScript localStorage API Intersection Observer CSS Variables Glassmorphism CSS Animations Responsive Design Google Fonts

🚀 CYBERSECURITY AWARENESS GOALS

01

Educate Before Attack

Most people learn about phishing after they've already been victimized. PhishGuard's goal is to educate users proactively, before they encounter a real attack, so they are mentally prepared to identify and avoid threats.

02

Simulate Real Scenarios

The quiz uses real-world phishing examples — actual URL patterns, SMS formats, and email tactics used by attackers — to give users practical experience recognizing threats in context.

03

Build Lasting Habits

Through repeated interaction with the quiz, threat analyzer, and learning modules, users build muscle memory for checking URLs, questioning urgency, and verifying requests — habits that protect them for life.

04

Accessible for Everyone

Cybersecurity education should not require technical knowledge. PhishGuard is designed in plain language with visual examples that anyone — students, seniors, professionals — can understand and apply immediately.

⚠️

ACADEMIC DISCLAIMER

PhishGuard is an academic front-end project created for educational purposes only. The threat intelligence scanner simulates URL risk analysis patterns and is not connected to a live threat database. For real threat detection, please use established tools like Google Safe Browsing, VirusTotal, or your organization's cybersecurity team.

All phishing examples shown are fictional recreations for educational purposes and do not represent any real attack targeting real organizations.
